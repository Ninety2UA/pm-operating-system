# SQL Query Explainer

## Context
Paste a complex SQL query and the tool breaks it down step-by-step — explaining each clause, visualizing the logical execution order, and suggesting optimizations. Bridges the gap between data engineers and business analysts. Supports goal: build & ship products, develop AI product skills.

Tech stack: React, Claude API for explanation generation, SQL syntax highlighter, step-by-step rendering.

## Scope
- SQL text input
- Claude-generated step-by-step breakdown with highlighted clauses
- Execution order diagram
- 1-2 optimization suggestions
