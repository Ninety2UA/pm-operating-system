# Webhook Event Viewer & Debugger

## Context
Real-time webhook inspector that displays incoming payloads in a structured, searchable format — useful for debugging API integrations, MMP postbacks, and ad platform callbacks. Built on MMP integration experience (Adjust, AppsFlyer). Supports goal: build & ship products, showcase technical depth.

Tech stack: React frontend, lightweight Node.js/Express backend for webhook receiver, WebSocket for real-time updates.

## Scope
- Unique webhook URL generator
- Real-time payload display (JSON formatted)
- Basic search/filter
- Test sender button
- Minimal backend
