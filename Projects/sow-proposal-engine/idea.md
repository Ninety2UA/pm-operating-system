# SOW & Proposal Template Engine

## Context
Fill in project details and the tool generates a professional Statement of Work or project proposal — complete with scope, deliverables, timeline, pricing, and terms. Directly useful for the consultancy. Supports goal: acquire clients for dbenger.com, build & ship products.

Tech stack: React, Claude API for dynamic section generation, HTML/PDF export, template system.

## Scope
- Form-based input (project type, scope, timeline, rate)
- Claude-generated SOW with 6 standard sections
- Clean HTML preview with PDF export option
