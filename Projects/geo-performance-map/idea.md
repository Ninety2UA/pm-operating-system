# Geo Performance Map

## Context
Interactive world map showing campaign performance by country/region — with color-coded CPI, ROAS, and volume metrics, plus drill-down to country-level detail. Built on multi-market campaign management experience across Northern Europe and EMEA. Supports goal: build & ship products, showcase analytics expertise.

Tech stack: React + D3 (topojson for map), sample data by country, tooltip/drill-down.

## Scope
- World map with 15-20 countries color-coded by selected metric (CPI, ROAS, volume)
- Metric selector dropdown
- Hover tooltip with country detail
