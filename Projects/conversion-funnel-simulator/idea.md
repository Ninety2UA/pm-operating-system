# Conversion Funnel Simulator

## Context
Define funnel stages and conversion rates, then simulate how changes at each stage (e.g., +10% landing page CVR) cascade through the full funnel — with revenue impact projections. Core to diagnostic work combining funnel analysis with marginal gain prioritization. Supports goal: build & ship products, showcase analytics expertise.

Tech stack: React + Recharts, slider inputs per stage, waterfall chart visualization.

## Scope
- 5-stage funnel (Impression -> Click -> Landing -> Lead -> Purchase)
- Editable conversion rates via sliders
- Waterfall chart
- "Revenue impact" summary card
