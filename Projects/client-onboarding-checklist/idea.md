# Client Onboarding Checklist Generator

## Context
Select a service type (analytics audit, campaign setup, AI implementation) and generate a customized onboarding checklist with prerequisites, timeline, and responsibility assignments. Systematizes client onboarding for scaling the consultancy. Supports goal: acquire clients for dbenger.com, build & ship products.

Tech stack: React, template-based generation, checklist UI with progress tracking, Tailwind.

## Scope
- 3 service types
- Each generating a 10-15 item checklist with owner (client/consultant)
- Timeline markers
- Progress tracker
