# ROAS vs. LTV Cohort Explorer

## Context
Interactive dashboard where users upload (or use sample) cohort data and explore how Day-7, Day-30, and Day-90 LTV curves shift across acquisition channels, geos, and campaign types — with breakeven analysis overlaid. Built on daily ROAS, CPI, LTV, and retention monitoring plus cohort/LTV views in BigQuery and Looker. LTV/cohort analysis is one of the most in-demand skills in growth and UA roles. Supports goal: build & ship products, become recognized expert.

Tech stack: React + Recharts or D3, Papaparse for CSV upload, Tailwind CSS.

## Scope
- Pre-loaded sample dataset (3 channels x 3 geos)
- Cohort curve visualization
- Breakeven line overlay
- Simple filter panel
- CSV upload as stretch goal
