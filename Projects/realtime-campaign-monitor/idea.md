# Real-Time Campaign Monitor (Mock)

## Context
Simulated real-time campaign monitoring dashboard that updates every few seconds — showing live spend, conversions, alerts, and a pacing indicator. Fully mock data but feels real. Brings the "war room" experience to portfolio. Supports goal: build & ship products, showcase analytics expertise.

Tech stack: React, setInterval for simulated updates, Recharts streaming charts, Tailwind.

## Scope
- 3 campaign cards with live-updating metrics
- Pacing progress bar
- Rolling 60-minute trend chart
- Color-coded alert badges
