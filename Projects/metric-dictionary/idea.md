# "Explain This Metric" Interactive Dictionary

## Context
Searchable, interactive dictionary of marketing, analytics, and AI metrics — each with a plain-English definition, formula, example, benchmark ranges, and "why it matters" context. SEO-friendly, shareable, and positions as an educator. Supports goal: build & ship products, become recognized expert.

Tech stack: React, static content (JSON/markdown), search/filter, Tailwind.

## Scope
- 30-40 metrics across marketing, analytics, and AI categories
- Each with: definition, formula, example, benchmark range, and one "pro tip"
- Search and category filter
