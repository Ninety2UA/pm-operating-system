# LLM Cost Calculator & Model Recommender

## Context
Describe your use case (task type, volume, latency needs, accuracy requirements) and the tool recommends the optimal LLM model + estimates monthly costs across providers (Anthropic, OpenAI, Google, open-source). Built on experience choosing Gemini Flash over Claude for knowledge base project specifically for cost efficiency. Shows AI thinking from a business perspective. Supports goal: develop AI product skills, build & ship products.

Tech stack: React, pre-built pricing/capability matrix (no API calls needed), decision logic, comparison table UI.

## Scope
- 5-6 question form (task type, tokens/day, latency, accuracy, budget)
- Recommendation engine with top 3 model suggestions
- Cost comparison table
- Brief rationale
