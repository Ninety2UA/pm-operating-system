# Tech Stack Decision Matrix

## Context
Input project requirements (scale, team size, budget, timeline, performance needs) and the tool recommends a tech stack — with tradeoff analysis and alternative options. Built on experience evaluating Vertex AI vs. Cloud Run, choosing between LLM providers for cost efficiency, and making pragmatic technology decisions. Shows architectural thinking valuable for technical leadership. Supports goal: develop AI product skills, build & ship products.

Tech stack: React, decision tree logic, comparison matrix UI, pre-built stack templates.

## Scope
- 6-8 requirement inputs
- 5 pre-built stack recommendations (startup, enterprise, AI-heavy, data-heavy, mobile)
- Comparison table with tradeoffs
- "Why this stack" explainer
