# Open-Source Contribution Tracker

## Context
Dashboard that aggregates and visualizes GitHub contributions, open-source projects, and technical writing — pulling from GitHub API and displaying activity patterns, languages used, and project impact. Built on published projects on GitHub (SKAN Reporting Pack). Shows active builder, not just a talker. Supports goal: build & ship products, become recognized expert.

Tech stack: React, GitHub API, Recharts for activity charts, Tailwind.

## Scope
- Contribution calendar heatmap
- Top 3-5 repos with star count and description
- Language breakdown chart
- Total contribution stats
