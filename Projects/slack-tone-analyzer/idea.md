# Slack Message Tone Analyzer & Rewriter

## Context
Paste a Slack message draft and the tool analyzes its tone (assertive, passive, unclear, too long) and offers 2-3 rewritten versions optimized for different outcomes (diplomatic, direct, urgent). Leverages cross-cultural communication experience. Supports goal: build & ship products, develop AI product skills.

Tech stack: React, Claude API, multiple output variants rendered side-by-side.

## Scope
- Text input for Slack message draft
- Tone analysis badge with confidence
- 3 rewritten variants labeled by strategy
- Word count comparison
