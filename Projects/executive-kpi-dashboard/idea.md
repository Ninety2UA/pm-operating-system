# Executive KPI Dashboard Template

## Context
Polished, interactive dashboard template for digital marketing KPIs — with real-time filtering, date range selection, and drill-down views. Pre-loaded with realistic sample data. Productized version of KPI scorecards and executive dashboards built at Google. Shows design for executives, not just analysts. Supports goal: build & ship products, become recognized expert.

Tech stack: React + Recharts, Tailwind, sample data JSON.

## Scope
- 4-6 KPI cards (Spend, Revenue, ROAS, CPI, CTR, Conversions)
- Date range picker
- Channel filter
- One drill-down chart (daily trend)
