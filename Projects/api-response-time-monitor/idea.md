# API Response Time Monitor

## Context
Configure a list of API endpoints, and the tool periodically pings them and displays response times, uptime percentages, and latency trends — with alerting thresholds. Shows operational maturity and full-stack thinking. Supports goal: build & ship products.

Tech stack: React + Recharts, lightweight backend (Node.js/cron) or client-side polling, Tailwind.

## Scope
- Add 3-5 endpoints via form
- Client-side polling every 60 seconds
- Response time chart per endpoint
- Red/yellow/green status indicator
