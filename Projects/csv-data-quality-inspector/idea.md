# CSV Data Quality Inspector

## Context
Upload a CSV and get an instant data quality report — missing values, type inconsistencies, outliers, duplicate rows, distribution summaries, and suggested cleaning actions. Demonstrates analytical rigor and data quality focus. Supports goal: build & ship products, build custom analytical solutions.

Tech stack: React, Papaparse, D3 for distribution charts, Tailwind.

## Scope
- CSV upload
- Summary stats per column (nulls, uniques, type detection)
- Top 3 issues flagged
- Mini distribution charts for numeric columns
