# Career Timeline Visualization (Enhanced)

## Context
Interactive, visual timeline of your career — with clickable milestones that expand to show achievements, skills acquired, and project highlights at each stage. More engaging than a static resume. Enhances existing career timeline on dbenger.com. Supports goal: build & ship products, personal branding.

Tech stack: React, D3 or CSS animation, Tailwind.

## Scope
- Timeline with 6-8 career milestones
- Expand-on-click with achievement bullets
- Skill tags per role
- Smooth scroll animation
