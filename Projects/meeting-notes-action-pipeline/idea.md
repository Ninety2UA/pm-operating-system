# Meeting Notes to Action Items Pipeline

## Context
A tool that takes raw transcript text (e.g., from Granola or Otter) and runs it through an LLM pipeline that extracts a meeting summary, structured action items (with owner and due date), and key decisions made. Optionally connect via webhook to push action items to Linear or Notion. Extremely practical and immediately understandable. Supports goals: **Develop AI product skills**, **Build & ship 10 products**.

## Scope
- Build transcript input interface
- Design LLM extraction pipeline (summary, actions, decisions)
- Implement structured output with owner/due date
- Add optional webhook integration (Linear/Notion)
- Deploy as web app
