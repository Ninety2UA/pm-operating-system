# Resume Tailoring Assistant

## Context
Upload your resume and a job description, and the tool highlights gaps, suggests rewording for better keyword alignment, and generates tailored bullet points — with a match score. Virally shareable — everyone job hunting would use this. Shows AI applied to a real, personal pain point. Supports goal: build & ship products, develop AI product skills.

Tech stack: React, Claude API, PDF.js for resume parsing (or text input), side-by-side diff view.

## Scope
- Two text areas (resume + JD)
- Claude API for analysis
- Rendered output with: match score, keyword gaps, 3-5 suggested rewrites
- No PDF parsing initially — just text input
