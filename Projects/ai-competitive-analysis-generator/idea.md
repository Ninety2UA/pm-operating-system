# AI-Powered Competitive Analysis Generator

## Context
Input a company name or URL, and the tool uses web search + Claude to generate a structured competitive analysis — covering positioning, pricing signals, strengths, weaknesses, and market context. Automates the kind of strategic research prepped before C-suite executive conversations at Google. Shows AI workflows that produce real business value, not just chat interfaces. Supports goal: develop AI product skills, build & ship products, acquire clients.

Tech stack: React, Claude API with web search tool, structured output parsing, clean report UI.

## Scope
- Text input for company name
- Claude API call with web search enabled
- Structured output rendered as clean report with 5-6 sections
- No persistent storage needed
