# QBR Report Generator

## Context
Tool that takes structured campaign performance data (CSV or form input) and generates a polished, client-ready Quarterly Business Review — complete with exec summary, trends, recommendations, and charts. Built on experience shipping automated QBRs and training 50+ reps on insight generation. Clearest "I would pay for this" signal from consulting clients. Recommended as #3 build priority. Supports goal: build & ship products, acquire clients, become recognized expert.

Tech stack: React frontend, Claude API for narrative generation, Recharts for embedded charts, PDF/HTML export.

## Scope
- Form-based input (or CSV upload) for key metrics
- Auto-generated exec summary (via Claude API)
- 2-3 chart types
- Clean HTML/PDF export
