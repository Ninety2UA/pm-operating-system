# Google Ads Budget Pacing Simulator

## Context
Interactive simulator where users input daily budgets, target ROAS, CPI goals, and pacing strategy — then watch a simulated 30-day campaign unfold with realistic variance, alerts, and optimization suggestions. Directly translates the "daily performance engine" run at Google, owning pacing and analytics across Google Ads for Northern Europe. Supports goal: build & ship products, become recognized expert.

Tech stack: React + Recharts for visualization, Tailwind CSS, no backend needed (client-side simulation logic).

## Scope
- Single campaign type (App Install)
- 30-day simulation with daily pacing chart
- Budget burn visualization
- 3-4 automated alert triggers (underspend, overspend, ROAS drop, CPI spike)
