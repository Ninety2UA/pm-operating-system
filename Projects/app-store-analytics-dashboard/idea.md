# App Store Analytics Dashboard

## Context
Dashboard that visualizes app store performance — ratings, reviews sentiment, download trends, keyword rankings — using sample data modeled on real app store analytics patterns. Shows breadth beyond paid acquisition into organic and store optimization. Supports goal: build & ship products, showcase analytics expertise.

Tech stack: React + Recharts, sample data, simple sentiment classification (keyword-based or Claude API).

## Scope
- Rating trend chart
- Review sentiment pie chart
- Download trend line
- Keyword ranking table
- All sample data
