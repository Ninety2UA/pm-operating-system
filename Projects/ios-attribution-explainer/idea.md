# iOS Attribution Explainer (Interactive SKAN Guide)

## Context
Interactive, visual explainer of how SKAdNetwork attribution works — covering conversion value schemas, timer windows, null values, and how advertisers should interpret SKAN data alongside MMP signals. Built by someone who developed the iOS SKAN Reporting Pack published at github.com/google/app-reporting-pack. SKAN expertise is rare and highly valued — positions as a thought leader. Supports goal: become recognized expert, build & ship products.

Tech stack: React with step-by-step animated diagrams (SVG or simple CSS animations), Tailwind.

## Scope
- 5-7 step walkthrough covering: install -> postback -> conversion value -> timer -> aggregation -> reporting
- Each step animated/interactive
- No backend
