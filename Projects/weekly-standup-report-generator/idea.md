# Weekly Standup Report Generator

## Context
Input your week's accomplishments, blockers, and plans — and the tool generates a polished standup report formatted for different audiences (team lead, executive, cross-functional partner). Demonstrates audience-aware communication design. Supports goal: build & ship products, develop AI product skills.

Tech stack: React, Claude API for multi-audience reformatting, template rendering.

## Scope
- Bullet-point input for accomplishments/blockers/plans
- 3 audience presets (team, executive, partner)
- Side-by-side output comparison
