# Multi-Market Export Opportunity Scorer

## Context
Input your app's current markets and performance data, and the tool scores untapped markets by opportunity size — factoring in CPI benchmarks, market size, competitive density, and language proximity. Built on experience driving 55% average YoY increase in market exports as Technical Apps Lead, consulting 25+ clients on international growth. Niche but high-signal differentiator. Supports goal: build & ship products, become recognized expert.

Tech stack: React, pre-loaded benchmark dataset, weighted scoring model, sortable table + map visualization.

## Scope
- 10-15 pre-loaded target markets
- Scoring algorithm based on 4-5 weighted factors
- Sortable results table
- Simple world map with color-coded scores
