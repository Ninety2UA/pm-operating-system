# Data Storytelling Slide Generator

## Context
Input key metrics and a narrative, and the tool generates a data storytelling slide (HTML) with chart, annotation, and headline — following best practices for executive presentations. Built on experience presenting strategic insights to C-suite executives. Bridges analytics and communication. Supports goal: build & ship products, become recognized expert.

Tech stack: React, Claude API for headline/narrative generation, Recharts for embedded chart, clean slide layout.

## Scope
- Form input (metric, trend direction, context, audience)
- Claude-generated headline + annotation
- Auto-selected chart type
- Styled slide preview with export
