# Knowledge Base Article Generator

## Context
Input a topic, context, and target audience, and the tool generates a structured knowledge base article with sections, FAQ, and metadata — following configurable style guidelines. Ties to Knowledge Base Automation system experience (Slack -> Claude -> Notion). Supports goal: build & ship products, develop AI product skills.

Tech stack: React, Claude API, markdown rendering, Tailwind.

## Scope
- Form with topic, context, audience, and tone inputs
- Claude-generated article with title, intro, 3-5 sections, FAQ, and metadata
- Rendered as styled markdown with copy/export
