# "Day in the Life" Campaign Optimizer Simulation

## Context
Narrative-driven simulation where the user plays as a UA manager for a mobile game. Each day brings new data, and you make optimization decisions (adjust bids, reallocate budget, pause creatives). Performance outcomes are simulated based on your choices. Extremely memorable and differentiated portfolio piece. Recommended as #4 build priority. Supports goal: build & ship products, showcase analytics and marketing expertise.

Tech stack: React, state management for game logic, Recharts for daily metrics, narrative text engine.

## Scope
- 7-day simulation (one week)
- 3 decision points per day (bid, budget, creative)
- Simulated outcomes with light randomness
- End-of-week performance scorecard
