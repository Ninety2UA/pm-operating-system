# Marketing Mix Allocation Visualizer

## Context
Drag-and-drop budget allocation tool where users distribute spend across channels (Search, Social, Programmatic, Video, App Install) and see projected outcome curves shift in real-time based on diminishing returns models. Supports goal: build & ship products, showcase analytics and product thinking.

Tech stack: React + Recharts, slider/drag inputs, diminishing returns curves modeled with log functions.

## Scope
- Slider-based allocation across 4 channels
- Real-time diminishing returns curve updates
- Summary card showing projected CPA/ROAS at current allocation
- No backend needed (client-side logic)
