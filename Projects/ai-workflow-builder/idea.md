# AI Workflow Builder (Visual)

## Context
Visual, node-based editor where users can chain AI operations — summarize -> extract -> classify -> format — and see data flow through each step in real-time. Think a mini n8n for LLM pipelines. Built on automation workflows with n8n, Make.com, and custom pipelines (Slack -> Claude -> Notion). Visually stunning portfolio piece showing AI orchestration understanding. Supports goal: develop AI product skills, build & ship products.

Tech stack: React, react-flow for node editor, Claude API for executing steps, Tailwind.

## Scope
- 4-5 pre-built node types (Input, Summarize, Extract, Classify, Output)
- Drag-and-drop canvas
- "Run" button that processes sample data through the chain
- Visible intermediate outputs
