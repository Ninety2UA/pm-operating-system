# Creative Performance Heatmap

## Context
Upload ad creative metadata (headlines, descriptions, CTAs, image tags) alongside performance metrics, and the tool generates a heatmap showing which creative elements correlate with higher CTR, CVR, and ROAS. Ties directly to creative analytics platform work and experience guiding creative optimizations for gaming and app advertisers. Supports goal: build & ship products, build custom analytical solutions.

Tech stack: React, D3 heatmap, Papaparse for CSV, basic statistical correlation logic in JS.

## Scope
- Pre-loaded sample dataset
- Heatmap grid (creative element x metric)
- Sortable columns
- Tooltip with sample size + confidence indicator
