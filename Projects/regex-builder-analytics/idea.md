# Regex Builder & Tester for Analytics

## Context
Interactive regex builder tailored for analytics use cases — UTM parameter extraction, event name parsing, URL pattern matching — with visual match highlighting and a library of common patterns. Supports goal: build & ship products, build custom analytical solutions.

Tech stack: React, regex engine (native JS), syntax highlighting, preset pattern library.

## Scope
- Regex input and test string input
- Real-time match highlighting
- 10 pre-built patterns for common analytics tasks (UTM extraction, GA event parsing, URL routing)
