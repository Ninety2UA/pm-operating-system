# Freelance Project Estimator

## Context
Input project parameters (type, complexity, deliverables, timeline) and get an effort estimate with a recommended price range — based on configurable rate models. Directly useful for the consultancy and shows product thinking. Shows business maturity alongside technical skill. Supports goal: acquire clients for dbenger.com, build & ship products.

Tech stack: React, form inputs, estimation algorithm, Tailwind.

## Scope
- Project type selector
- Complexity slider
- Deliverable checklist
- Calculated hours estimate
- Price range (based on configurable hourly rate)
