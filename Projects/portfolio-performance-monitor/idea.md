# Portfolio Site Performance Monitor

## Context
Monitoring page for your own portfolio site that tracks Core Web Vitals, uptime, SEO scores, and performance metrics — with trend charts and optimization recommendations. Meta and self-referential — shows continuous improvement mindset. Supports goal: build & ship products, showcase analytics expertise.

Tech stack: React, Lighthouse API (or PageSpeed Insights API), Recharts for trend charts.

## Scope
- Lighthouse score cards (Performance, Accessibility, SEO, Best Practices)
- One trend chart showing score over time
- 3-5 top optimization recommendations
