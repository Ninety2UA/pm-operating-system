# LLM Cost Calculator & Model Recommender

## Context
Describe your use case (task type, volume, latency needs, accuracy requirements) and the tool recommends the optimal LLM model and estimates monthly costs across providers (Anthropic, OpenAI, Google, open-source). Helps teams make data-driven decisions about which model to use for different workloads.

Tech stack: React, pre-built pricing/capability matrix (no API calls needed), decision logic, comparison table UI.

## Scope
- 5-6 question form (task type, tokens/day, latency, accuracy, budget)
- Recommendation engine with top 3 model suggestions
- Cost comparison table
- Brief rationale
